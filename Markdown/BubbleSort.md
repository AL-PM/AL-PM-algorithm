# BubbleSort
---
## 알고리즘 설명

---
### 문제
$N$개의 수가 주어졌을때, 이를 오름차순으로 정렬하여 출력하라.

### 입력
첫째 줄에 수의 개수 $N$ $(1 ≤ N ≤ 1000)$이 주어진다.

둘째 줄에는 공백 한칸으로 구분된 정수 $x_n$ $(-1000 ≤ x_n ≤ 1000)$가 주어진다. 

### 출력
첫째 줄에 정렬된 $N$개의 수를 공백 한칸으로 구분해서 출력한다.

---
### 예제 입력
```
7
3 1 4 1 5 9 2
```

### 예제 출력
```
1 1 2 3 4 5 9
```
